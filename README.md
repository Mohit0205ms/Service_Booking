# Servcie Booking App

This is a servcie booking app built with React Native and TypeScript using Expo.

## Features
- Login screen with basc authentcation
- List of services
- Service details
- Booking form
- Bookings list with locl storage

## Installation
1. Install dependencies: npm install
2. Run the app: npx expo start

## Libraries Used
- React Native
- Expo
- @react-navigation/native
- @react-navigation/stack
- @react-navigation/bottom-tabs
- @react-native-async-storage/async-storage

## Assumptons
- No backend requred for this prototype
